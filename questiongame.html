What is the Javascript here doing?
I have seven small pieces of code here that make up my question game.

One

The first two lines of code are variables I am declaring here to keep a counter for the number of question the user gets right, and a string that I will be adding to later to give the user a response unique to the questions the user answered incorrectly

Two

Here I defined a new object that I will use to store my questions and the associated properties, as well as a method used to compare the user response to the intended answer of the question. This method also adds to my counter in order to keep track of the number of correct answers the user has accumulated.

Three

I created three new objects of the type created above.

Four

I created an array and populated it with the objects I created above in order to have those objects associated with an index number.

Five

This is an alert greeting the user.

Six

This is a for loop that cycles through my array and, for each object, prompts the question, defines the user answer. The loop then runs the method created earlier in order to determine if the user was correct and to run an alert notifying the user. This loop runs through my entire array, asking all questions stored there.

Seven

This final alert uses the variable counter I had created earlier to notifying the user about the number of questions they answered correctly. It also prompts an alert with a variable holding a response to each question answered incorrectly. This alert is held in an if statment. If the user got less than 100%, the first alert displays. If they got them all right, the else statement alert displays.
